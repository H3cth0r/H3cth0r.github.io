Projects

These are some of my projects, please hover over the information, so the images get presented:


  • Flight Tracker Data Processor

    This is a small program that scrapes flight data of a selected flight. With this program we can gather the whole log data and registered points of a flight. The reason I created this program, is por gathering a robust data set, for using machine learning and creating routes for small airplanes, depending of maximum high, velocity, power, etc. The machine learning part is still on development. This is just a part of a bigger personal project.

  • Aerodromes locations in México

    I developed this program last year, it is intended to present all available aerodromes or landing tracks in México, or just the registered one. For this I downloaded a database of all these aerodromes. I used javascript and node.js for the development, and it was presented on an openstreetmap map. This project can still be optimized, for grouping the aerodromes depending on the zone in México.

  • TellMeWhoseThere

    This is a python project that lets me register and know who is logged into my home network. I implemented this on a raspberry pi. This way I can have a registration on my network of who, when and how much time the user was logged into my local-network.

  • 3D representation of F1-Matlab

    This was a project I developed in my first semester for my physics subject, where I had to develop a 2d animation of a car, moving on a track. The interesting thing about this project is that we developed a 3D animation, and created structures with graphs. I basically developed a basic game engine.
